# app/

React Native (TypeScript) mobile app. Will hold the four client-side modules once implementation starts:

- `voice/` — speak-to-respond conversation loop (STT → Claude → TTS), English + Arabic first
- `navigation/` — maps, directions, geofencing around the holy sites
- `guide/` — historical/spiritual content triggered by location or voice query
- `health/` — symptom tracking, wearable integration (Apple Watch/HealthKit, etc.), emergency escalation UI

No code yet — see `/CLAUDE.md` at the project root for full context.
